## 💻 Use Immediately

### Chat with AI
```python
from chatbot.chatbot import Chatbot
bot = Chatbot(name="NEURON")
print(bot.chat("Hello! What can you do?"))
```

### Automate Tasks
```python
from agent.agent import Agent
agent = Agent()
result = agent.execute_task("Calculate 100 + 50")
print(result["final_result"])
```

### Generate Code
```python
from code_generator.generator import CodeGenerator
gen = CodeGenerator()
code = gen.generate("Create a hello world function")
print(code["code"])
```
