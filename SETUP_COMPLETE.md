## 💡 Quick Code Examples

### Use Chatbot
```python
from chatbot.chatbot import Chatbot

bot = Chatbot(name="NEURON")
response = bot.chat("Tell me about AI")
print(response)
```

### Use Agent
```python
from agent.agent import Agent

agent = Agent()
result = agent.execute_task("Calculate 100 * 5")
print(result["final_result"])
```

### Use Code Generator
```python
from code_generator.generator import CodeGenerator

gen = CodeGenerator()
code = gen.generate("Create a function that reverses a string")
print(code["code"])
```
